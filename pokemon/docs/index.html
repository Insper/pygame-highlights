Pokemon Gym

Descrição

O jogo, inspirado em Pokémon, permite que um personagem, munido de um deck de Pokémon, explore uma ilha, um Centro Pokémon e um Ginásio. O estilo de batalha em turnos é o centro da experiência, onde os treinadores lideram seus Pokémon em ataques alternados por turno. Cada Pokémon possui uma variedade de movimentos, e os jogadores escolhem estrategicamente um para cada turno. Além disso, os tipos de Pokémon desempenham um papel crucial na eficácia dos ataques, adicionando uma camada extra de estratégia ao jogo. Reúna seus conhecimentos sobre Pokémon e embarque nessa jornada emocionante!

Como Jogar

Para jogar, é necessário ter o Python 3 instalado na máquina. Além disso, se você estiver no Windows, consulte o guia abaixo para mais informações.

Após instalar o python, abra um terminal e execute o comando 'pip install pygame' para baixar a biblioteca necessária, após execute o comando 'cd caminho': no terminal , substituindo a palavra 'caminho' pelo path da pasta onde está localizado o arquivo do jogo, por fim escreva 'python arquivo' substituindo 'arquivo' pelo nome do arquivo do jogo sendo o arquivo 'jogo.py, para poder jogar o jogo. O jogo é jogado com as seguintes teclas:

Movimento: w/a/s/d.

Ações em batalha: teclas da seta / esc / enter.

Outros: algumas interações são feitas com o mouse bottom 1.

Jogando no Windows

No Windows é necessário instalar algumas dependências adicionais. O módulo utilizado por este projeto pode não funcionar corretamente com o prompt de comando padrão do Windows. Por esse motivo, comece instalando o Windows terminal.
